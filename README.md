# BankApplication

**Description:**  
Java-based Bank Application integrated with MySQL. Supports account creation, deposits, withdrawals, and balance check. Demonstrates JDBC usage, SQL operations, and simple menu-driven console interface.

## Features
- Create new accounts
- Deposit and withdraw money
- Check account balance
- Persistent storage using MySQL

## Setup
1. Install Java JDK and MySQL
2. Create database `bankdb` and table `accounts`
3. Place `mysql-connector-j-9.5.0.jar` in project folder
4. Compile:
   ```bash
   javac -cp ".;.\mysql-connector-j-9.5.0.jar" main.java
